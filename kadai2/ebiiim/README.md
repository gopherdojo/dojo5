# imgconv

## Usage

```text
Usage:
  imgconv [-source_ext=<ext>] [-target_ext=<ext>] DIR
Arguments:
  -source_ext=<ext>     source extension (jpg, png, tiff, bmp) [default: jpg]
  -target_ext=<ext>     target extension (jpg, png, tiff, bmp) [default: png]
```

## Development

### Build

```bash
make build
```

## Test
